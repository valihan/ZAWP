# ZAWP
Массовое подтверждение входящих АВР


Просто подтверждение входящих АВР на портале ЭСФ РК.

Для работы у пользователя должна быть настройка в таблице FIKZ_DI_V_CUST - для работы с ЭСФ.
Дополнительные настройки - в SOAMANAGER
1. Для веб-сервиса ZAWP_CO_AWP_WEB_SERVICE указать url https://esf.gov.kz:8443/esf-web/ws/api1/AwpWebService
2. Для веб-сервиса ZAWP_CO_LOCAL_SERVICE указать url подключения к серверу подписи

   Для работы с сервером подписи используется Z-овский ZAWP_CO_LOCAL_SERVICE, так как раньше в стандартном классе была какая-то проблема с подписью XML. Возможно, надо переделать на стандарт
